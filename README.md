🚀 Project Showcase: E-Visa Follow-Up Processing System

As part of my Web Engineering coursework, I built a full-stack visa application portal that simulates a government-style immigration system — from application submission to admin-side approval workflows.

🔧 Tech Stack
• Backend: Node.js + Express.js (MVC architecture)
• Database: MySQL (with connection pooling via mysql2)
• Authentication: JWT + bcrypt for secure password hashing
• File Handling: Multer for document uploads (PDF/JPG/PNG)
• Security: Helmet, CORS, express-rate-limit, express-validator
• Logging: Winston (structured logs for errors, security events & performance)
• Frontend: HTML, CSS & vanilla JavaScript

✨ Key Features
• Secure user registration & login
• Visa application submission with auto-generated reference numbers
• Document upload with type/size validation
• Real-time application status tracking (Submitted → In Review → Approved/Rejected)
• Automated in-app notifications on status changes
• Admin dashboard for managing and updating applications

🔮 Planned Future Improvements
• Email/SMS notifications for status updates
• Password reset functionality
• National ID/CNIC verification for stronger identity checks
• Online payment gateway integration
• Pagination & server-side filtering for scalability
• Migration to React for a more dynamic frontend
• Automated testing suite

This project gave me hands-on experience with full-stack development, REST API design, authentication flows, and building security-conscious backend systems.
